# tauan-games


Repositório onde armazenaremos Jogos que criamos com Vibe Coding.

## Jogos

Teremos algumas arquitetura diferentes para construir jogos.

1. Arquitetura de usa html + css puro e Java Script vanilla para construir jogos simples
2. Arquitetura que usa Golang para construir Jogos mais complexos.


### Jogo 1:

Jogo do dinossauro que pula cactos. Será similar ao jogo que tem no google chrome quando a internet fica indisponível. O jogo será construído usando html + css puro e Java Script vanilla.
- Deve ter um dinossauro mais bem ilustrado do que o do google chrome.
- O chão deve ter uma cor marron alaranjada e o cel deve conter:
    - Durante a noite teremos um céu estrelado e lua.
    - durante o dia teremos o sol e nuvens.
    - O ceu (fundo) 'eestático.
