# Tech-Stack Memory: tauan-games

## Stack (a definir por jogo)

| Categoria | Opções consideradas | Notas |
|---|---|---|
| Engine 2D web | Phaser.js, Kaboom.js | jogos no browser |
| Engine 2D desktop | Pygame | Python, fácil para iniciantes |
| Engine 3D | Three.js, Babylon.js | experimentos 3D web |
| Linguagem | JavaScript / TypeScript, Python | conforme engine escolhida |

## Padrão de Deploy

- Jogos web: GitHub Pages ou Netlify (estático, sem custo)
- Jogos desktop: executável buildado e linkado no README
