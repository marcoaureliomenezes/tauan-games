# Product Memory: tauan-games

## O que é

Repositório de jogos e experimentos interativos — espaço pessoal de aprendizado e criação de Tauan.

## Para quem

Uso pessoal e aprendizado. Pode servir como portfólio de desenvolvimento de jogos no futuro.

## Jogos / Projetos Atuais

| Projeto | Pasta | Descrição | Status |
|---|---|---|---|
| Tauan T-Rex | `tauan-trex/` | Enhanced Chrome Dino clone personalizado para Tauan — pixel art, high score, day/night | Approved + Implementado |
| Aero Fighters (Aero Strike) | `aero-fighters/` | F-35 Lightning II Ground Strike — Three.js, ataque a alvos militares estáticos. Codename `aero-fighters` mantido por inércia de paths; nome visível ao jogador = "Aero Strike" | Approved + Implementado |
| Testing Infrastructure | `tests/` | Playwright smoke quality gate para todos os jogos | Approved + Implementado |

## Público-alvo principal

Tauan (filho do operador, criança). Jogos devem ser imediatamente jogáveis, sem tela de loading, sem erros, com controles simples.

## Objetivos

- Criar jogos realmente jogáveis para Tauan
- Capturar a experiência de jogos de referência (Chrome Dino, Aero Fighters Assault N64)
- Todo jogo deve passar em testes Playwright automatizados antes de ser entregue ao operador para revisão
- Publicar jogos jogáveis online (GitHub Pages) quando maduros
